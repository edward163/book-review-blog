# Book Review Blog

This repository is being converted into a static blog (book reviews).

I will create a `static-site` branch with a simple static website (HTML/CSS) and an example post. After that you can enable GitHub Pages in repository Settings -> Pages and set the source to the `static-site` branch.

If you'd prefer a specific static site generator (Jekyll, Hugo, Eleventy) or GitHub Actions deployment, tell me and I will set that up instead.
